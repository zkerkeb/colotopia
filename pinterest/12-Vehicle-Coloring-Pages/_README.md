# Vehicle Coloring Pages

**Pinterest Board:** Vehicle Coloring Pages
**Number of pins:** 15

## How to pin

1. Open pinterest.com and go to the matching board
2. Click '+' then 'Create Pin'
3. Upload the image from this folder
4. Copy Title and Description from _PIN-INSTRUCTIONS.csv
5. Paste the Page Link
6. Publish
